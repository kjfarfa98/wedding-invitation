# デザイン仕様

## コンセプト

青・白を基調、ゴールドをアクセントに使ったシックで落ち着いた雰囲気

## カラーパレット

| 変数名 | 値 | 用途 |
|---|---|---|
| `--deep-ocean` | `#03045e` | ヒーロー背景（最暗部）・フッター背景 |
| `--ocean` | `#0077b6` | メインカラー・セクション見出し・RSVP背景 |
| `--turquoise` | `#00b4d8` | アクセント・カード上辺ボーダー・タイムラインドット |
| `--sky` | `#90e0ef` | グラデーション薄め |
| `--foam` | `#e8f6fb` | セクション背景（偶数）・フォームボックス |
| `--white` | `#ffffff` | ベース背景 |
| `--gold` | `#c9a227` | 区切り線・オーナメント |
| `--gold-light` | `#e8d5a3` | RSVPセクション内オーナメント |
| `--text-dark` | `#1a2f4e` | 本文テキスト |
| `--text-mid` | `#2c5f7a` | サブテキスト |
| `--text-light` | `#6b98b5` | 補足テキスト・セクション日本語サブタイトル |

## タイポグラフィ

| フォント | 用途 |
|---|---|
| Noto Serif JP (300/400/600) | 本文・日本語全般 |
| Playfair Display (400/600/italic) | 英語セクション見出し・ヒーロー日付 |
| Great Vibes | ご挨拶署名・フッター名前 |

## ページ構成・セクション

| セクションID | 背景色 | 内容 |
|---|---|---|
| `#hero` | deep-ocean → ocean → turquoise → sky グラデーション | 日付・名前・スクロールヒント |
| `#greeting` | white | ご挨拶文 |
| `#details` | foam | 挙式・披露宴カード |
| `#schedule` | white | タイムライン |
| `#dresscode` | foam | ドレスコードカード |
| `#rsvp` | ocean → turquoise グラデーション | RSVPフォームリンク |
| `#access` | white | 会場情報・地図 |
| `footer` | deep-ocean | 名前・日付・連絡先 |

セクション間はSVGウェーブ区切り線（`.wave-sep`）で接続する

## UIコンポーネント

### セクション見出し構造

```
.sec-en   → 英語タイトル（Playfair Display、ocean色）
.sec-ja   → 日本語サブタイトル（text-light色）
.ornament → ゴールド横線 + ✦アイコン
```

### カード（`.detail-card`、`.dresscode-card`）

- 背景：white、角丸14px
- 上辺：3px turquoise ボーダー
- シャドウ：`0 4px 22px rgba(0,119,182,.1)`
- ホバー：上方向6px移動 + シャドウ強調

### タイムライン（`#schedule`）

- 中央縦線：turquoise → sky グラデーション
- ドット：turquoise、白ボーダー
- 奇数行：右側テキスト、偶数行：左側テキスト
- モバイル（≤640px）：全て左側テキストに切り替え

### ボタン（`.btn`）

- 背景：white、テキスト：ocean色
- 角丸：50px（pill形）
- ホバー：上方向3px移動 + シャドウ強調

### アクセスグリッド（`.access-grid`）

- 2カラムグリッド
- `.map-wrapper`は全幅（`grid-column: 1 / -1`）
- モバイルは1カラム

## 装飾要素

### ヒーローセクション

- **ハイビスカス透かし**（SVG）: 四隅に配置、opacity 13%、白色
- **バブルパーティクル**: JSで22個生成、下から上へアニメーション
- **スクロールヒント**: 下部中央、バウンスアニメーション
- **ウェーブ区切り**: 下辺にSVGウェーブ（白）

### フッター

（装飾なし）

## アニメーション

| 名前 | 対象 | 内容 |
|---|---|---|
| `bubble-rise` | .bubble | 下から上へ浮き上がり |
| `hint-bounce` | .scroll-hint | 上下バウンス（2.2s周期） |
| `.fade-in` + IntersectionObserver | 各セクション要素 | スクロール時にフェードイン＋上方向スライド（0.85s） |

フェードインのdelayクラス：`.delay-1`（0.15s）、`.delay-2`（0.3s）、`.delay-3`（0.45s）

## レスポンシブ

- コンテナ最大幅：820px
- セクションpadding：88px 24px
- ブレークポイント：640px（タイムライン・アクセスグリッドのレイアウト変更）
- フォントサイズは`clamp()`で流動的に変化
